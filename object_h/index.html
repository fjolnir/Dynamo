_Obj_guts

typedef struct { 
    Class_t *isa; 
    long referenceCount; 
} _Obj_guts;
Discussion

The core of an object, contains a reference to it's class & it's reference count
Add OBJ_GUTS at the beginning of a struct type in order to make it a valid, retainable object
