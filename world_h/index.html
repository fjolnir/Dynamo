_WorldEntity

struct _WorldEntity { 
    OBJ_GUTS World_t *world; // Weak reference 
    Obj_t *owner; // Weak reference (Owner should retain the entity) 
    cpBody *cpBody; 
    LinkedList_t *shapes; 
    WorldEntity_UpdateHandler updateHandler; 
    WorldEntity_CollisionHandler preCollisionHandler; 
    WorldEntity_CollisionHandler collisionHandler; 
    WorldEntity_CollisionHandler postCollisionHandler; 
};
Fields
world

The world which contains the entity

owner

An arbitrary pointer to the "owner" of the entity (Useful to attach the entity to for example a sprite)

updateHandler

A function that is called every time the entity's state changes

preCollisionHandler

A function that is called right before the entity collides

collisionHandler

A function that is called during a collision of the entity

postCollisionHandler

A function that is called when the entity separates from whatever entity it was in collision with

Discussion

A Game entity is an object that can be rendered and/or included in the physics simulation
