util_pathForResource

extern bool util_pathForResource(
    const char *name,
    const char *ext,
    const char *dir,
    char *output,
    int maxLen);
Parameters
name

The file name of the resource

ext

The file extension of the resource

dir

The directory of the resource (relative to the game resource directory)

output

The output buffer

maxLen

The length of the output buffer

Discussion

Returns the filesystem path for a given resource.
