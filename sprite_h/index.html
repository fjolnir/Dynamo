Sprite

Includes:
"object.h"
"renderer.h"
"texture_atlas.h"

Introduction

Sprite class supporting atlases with multiple animations consisting of multiple frames

Y Offset in atlas: Animations, X Offset: Frames

Sprite_t

typedef struct _Sprite { 
    OBJ_GUTS RENDERABLE_GUTS TextureAtlas_t *atlas; 
    vec3_t location; 
    vec2_t size; 
    float scale, angle; 
    bool flippedHorizontally; 
    bool flippedVertically; 
    int activeAnimation; // The y offset of the active animation 
    SpriteAnimation_t *animations; 
} Sprite_t;
Fields
atlas

The sprite's texture atlas

location

The location at which the sprite is drawn in space.

size

The size of the sprite.

scale

The scale at which the sprite is drawn.

angle

The angle at which the sprite is drawn.

flippedVertically

Indicates whether or not the sprite is drawn flipped over the X axis.

flippedHorizontally

Indicates whether or not the sprite is drawn flipped over the Y axis.

activeAnimation

Indicates the active animation (0 being the bottom animation in an atlas)
